# Fakerni — Project Report (LaTeX / Overleaf)

This folder contains the end-of-studies report (rapport) as a self-contained
LaTeX project.

## Compile on Overleaf
1. Go to [overleaf.com](https://www.overleaf.com) → **New Project** → **Upload Project**.
2. Zip this `report/` folder (so the zip contains `rapport.tex` and the
   `images/` subfolder) and upload it — or drag the files in.
3. Overleaf will detect `rapport.tex` as the main file. Set the compiler to
   **pdfLaTeX** (Menu → Compiler) if it isn't already, then click **Recompile**.

## Before you submit — edit your details
Open `rapport.tex` and change the values near the top (the `\my...` commands):
`\myAuthor`, `\mySupervisor`, `\myUniversity`, `\myDept`, `\myYear`.

## Notes
- Figures live in `images/`. If an image is missing, the document still compiles
  and shows a placeholder box in its place.
- It uses only standard packages that Overleaf ships with by default.
