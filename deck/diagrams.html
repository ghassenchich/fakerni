Fakerni — System Architecture
Three-tier, real-time; stateless API
Web Client
React 19 · Vite · Tailwind v4
Mobile Client
React Native · Expo
▲  REST over HTTPS  +  WebSocket over WSS  (JWT auth)  ▲
Django + DRF + Channels  (ASGI / Daphne)
REST API · WebSocket consumers · JWT auth middleware · business logic · AI orchestration
▲
PostgreSQL
Relational data store
Redis
Channel layer (fan-out)
Google Gemini
AI parsing (external)
Firebase FCM
Push notifications (external)
Why this shape: the API is stateless (JWT), so it scales horizontally behind a load balancer; Redis fans real-time events across instances; PostgreSQL holds all persistent data; external services (Gemini, FCM) are called on demand and degrade gracefully if unavailable. Everything is containerised (Docker) and verified by CI on every push.
